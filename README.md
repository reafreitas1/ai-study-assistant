# AI Study Assistant 🧠💡

A powerful Full-Stack application that transforms any topic or long text into structured summaries and interactive 3D flashcards using Generative AI.

## 🚀 Technologies
- **Backend:** Java 21, Spring Boot 3.3, Spring AI (Google Gemini API).
- **Frontend:** Angular 18, TypeScript, Signals, CSS3 (3D Transforms).
- **Architecture:** REST API with structured JSON exchange.

## ✨ Features
- **Smart Summarization:** Automatically extracts the most important points from any content.
- **Dynamic Flashcards:** Generates a custom number of interactive study cards (1-15).
- **Language Detection:** Automatically detects and responds in the user's language.
- **Active Recall UI:** 3D flip-cards designed for efficient memorization.

## 🛠️ How to Run
1. **Backend:**
   - Set your Gemini API Key as an environment variable: `GEMINI_API_KEY`.
   - Run `./mvnw spring-boot:run` inside `iastudy-back`.
2. **Frontend:**
   - Run `npm install` and `ng serve` inside `iastudy-front`.
   - Access `http://localhost:4200`.
